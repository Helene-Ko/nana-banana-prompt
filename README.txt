Nana Banana Prompt Constructor
==============================

Структура проекта:
- index.html  : разметка страницы
- styles.css  : стили интерфейса
- app.js      : логика генерации промпта

Запуск:
1) Откройте файл index.html в браузере.
2) Выберите параметры баннера.
3) Нажмите "Скопировать промпт" и вставьте текст в nana-banana.ai.
